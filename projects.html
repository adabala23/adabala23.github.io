🧠 AI-Powered Retinal Disease Detection System

The primary objective of this project was to develop an intelligent healthcare solution capable of accurately identifying retinal diseases from medical fundus images. Early detection of retinal disorders is critical in preventing vision loss, and this project aimed to assist healthcare professionals by providing an AI-based diagnostic support system.

To achieve this, multiple state-of-the-art deep learning architectures including DenseNet201, ResNet101 and EfficientNetV2L were integrated using a Hybrid Ensemble CNN approach. Extensive image preprocessing techniques such as resizing, normalization, augmentation and noise reduction were applied to improve model generalization and performance.

Transfer learning was utilized to leverage pre-trained models, significantly reducing training time while improving accuracy. Grad-CAM visualization techniques were implemented to enhance model interpretability and provide explainable AI outputs, allowing users to understand which retinal regions influenced predictions.

The final model achieved approximately 85% classification accuracy across multiple retinal disease categories and demonstrated the effectiveness of deep learning in automated medical image analysis and healthcare decision support systems.

Role: Data Collection, Image Preprocessing, Model Development, Training, Evaluation and Performance Analysis.

Domain: Healthcare AI & Deep Learning

Technologies Used: Python, TensorFlow, Keras, OpenCV, Deep Learning, CNN, Transfer Learning, Grad-CAM.

🏥 Medi Trust – AI Driven Medical Crowdfunding Platform

Medi Trust is an AI-powered transparent medical crowdfunding platform designed to address fraud, lack of transparency and trust issues commonly observed in online medical fundraising campaigns. The system ensures that donations are utilized only for genuine medical purposes through intelligent verification mechanisms.

The platform incorporates Optical Character Recognition (OCR) and Natural Language Processing (NLP) techniques to automatically extract and validate information from uploaded medical documents. Multiple verification layers were implemented to identify inconsistencies, document tampering and fraudulent submissions.

An AI-powered story generation module was developed to create clear and compelling campaign descriptions based on verified medical information. Additionally, a phased fund release mechanism was implemented to ensure that payments are transferred directly to verified hospitals instead of individual accounts.

The project significantly improves transparency, donor confidence and accountability while reducing the risk of misuse of crowdfunding resources.

Role: Frontend Development, Backend Integration, Database Design, Verification Logic and System Testing.

Domain: Healthcare Crowdfunding & Artificial Intelligence

Technologies Used: React.js, Node.js, Python, MySQL, OCR, NLP, REST APIs.

🚗 Car Accident Analysis & Risk Prediction

The objective of this project was to analyze large-scale accident datasets and identify hidden patterns that contribute to road accidents and fatalities. The project focused on understanding accident trends, regional risk factors and vehicle-related influences using machine learning techniques.

The dataset was cleaned, transformed and prepared for analysis using various data preprocessing methods. Exploratory Data Analysis (EDA) was performed to discover meaningful trends and relationships among different accident-related attributes.

Machine learning clustering algorithms were applied to group locations based on accident severity, vehicle involvement and fatality rates. These clusters helped identify high-risk regions and enabled deeper understanding of accident-prone areas.

The generated insights can support policymakers, transportation authorities and safety organizations in implementing preventive measures and improving road safety infrastructure.

Role: Data Cleaning, Data Analysis, Feature Engineering, Clustering and Visualization.

Domain: Data Analytics & Machine Learning

Technologies Used: Python, Machine Learning, Clustering, Data Analytics, Pandas, NumPy and Matplotlib.

🌦 Real-Time Weather Forecasting System

The objective of this project was to develop a weather forecasting application capable of providing accurate and real-time weather information for any location. The application integrates external weather services through REST APIs and dynamically fetches environmental data based on user input.

The system processes JSON responses received from the API and displays important weather parameters such as temperature, humidity, atmospheric pressure, wind speed and weather conditions in a user-friendly format.

Location-based search functionality was implemented to allow users to retrieve weather information for different cities and regions instantly. The project improved my understanding of API integration, JSON parsing, exception handling and real-time data processing.

Role: API Integration, Data Processing, User Interface Development and Testing.

Domain: API Integration & Software Development

Technologies Used: Python, REST APIs, JSON.
